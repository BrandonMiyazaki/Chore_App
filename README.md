# ChoreApp

ChoreApp is an application designed to help households and families manage and track chores. Users earn points for completing assigned chores, making household responsibilities more engaging and rewarding.

## About

ChoreApp allows you to:

- **Track Chores** – Create, assign, and manage a list of household chores.
- **Earn Points** – Users accumulate points each time a chore is marked as completed.
- **Motivate Participation** – The points system encourages everyone in the household to contribute.
